
# 🚀 Mi Portafolio Personal - Kelvin Chirinos

¡Bienvenido al repositorio de mi portafolio! Este es el lugar donde muestro mi trayectoria como desarrollador, los proyectos en los que he trabajado y las tecnologías que domino.

El objetivo de este proyecto es tener una plataforma centralizada y visualmente atractiva para conectar con otros desarrolladores y posibles reclutadores.

---

## ✨ Características Principales

- **Diseño Responsivo:** Optimizado para dispositivos móviles, tablets y escritorio.
- **Modo Oscuro/Claro:** Implementado para una mejor experiencia de usuario.
- **Sección de Proyectos:** Galería detallada con enlaces a demos en vivo y repositorios.
- **Formulario de Contacto:** Integración funcional para recibir mensajes directamente.
- **Optimización SEO:** Configurado para ser fácilmente indexado por buscadores.

## 🛠️ Tecnologías Utilizadas

Este proyecto fue construido utilizando las siguientes herramientas:

* **Frontend:** HTML5/CSS3
* **Estilos:** Tailwind CSS
* **Despliegue:** Vercel
* **Otras:** Astro
